# 贔貅记账导入方案总结

## 📋 设计概览

为你的 Rails 记账应用设计了一套完整的贔貅记账数据导入方案，支持 **22,000+ 条交易记录**。

---

## 🎯 核心特性

### 1. 双通道导入
- **Web UI**：4步向导式导入（推荐用户使用）
- **命令行**：Rake 任务（推荐批量导入）

### 2. 智能映射
- ✅ 自动识别账户（50+ 种）
- ✅ 自动识别分类（20+ 种）
- ✅ 可配置的映射关系
- ✅ 支持自定义调整

### 3. 数据处理
- ✅ 转账双向记录（出账 + 入账）
- ✅ 空分类自动归类
- ✅ 金额精度保留（2位小数）
- ✅ 日期格式验证

### 4. 错误处理
- ✅ 单条失败不影响整体
- ✅ 详细的错误日志
- ✅ 导入统计报告
- ✅ 预览和验证机制

## 🚀 使用方式

### 方式1：Web UI（推荐）
```
1. 访问 http://192.168.10.232:3000/imports/pixiu
2. 上传 贔貅记账_所有交易流水.csv
3. 预览数据统计
4. 配置账户/分类映射
5. 开始导入
```

### 方式2：命令行
```bash
cd ~/Desktop/ledger
bundle exec rake import:pixiu FILE=/path/to/pixiu_export.csv
```

### 分类映射（自动创建）
```

```

```
  农行2917:     -1000 (EXPENSE, 转账)
  支付宝余额:   +1000 (INCOME, 转账)
```

## 📈 导入流程

```
上传文件
   ↓
验证格式 (CSV, UTF-8, <50MB)
   ↓
预览数据 (统计有效/无效/转账)
   ↓
配置映射 (账户和分类)
   ↓
开始导入 (逐行处理)
   ↓
显示结果 (导入统计)
```
